# [Feature] 로그인 UI 구성

## User Story
As a 사용자,
I want 로그인 화면에서 이메일/비밀번호를 입력하고 로그인 버튼을 누르고 싶다,
So that 로그인을 시도할 수 있다.

---

## Acceptance Criteria
- [ ] 이메일/비밀번호 입력 필드가 있다.
- [ ] 이메일 형식이 아니면 안내 메시지가 보이거나 버튼이 비활성화된다.
- [ ] 비밀번호가 8자 미만이면 안내 메시지가 보이거나 버튼이 비활성화된다.
- [ ] 로그인 버튼을 탭하면 “로그인 시도” 이벤트가 발생한다(실연동은 다른 Issue).

## Edge Cases
- 중복 탭(연타)
- 키보드 포커스/입력 중 상태

---

## How to Test
1. 로그인 화면 진입 → 이메일/비밀번호 입력 필드가 보이는지 확인
2. 이메일 형식이 아닌 값 입력 → 안내/버튼 비활성 동작 확인
3. 비밀번호 8자 미만 입력 → 안내/버튼 비활성 동작 확인
4. 정상 입력 → 로그인 버튼 활성 확인

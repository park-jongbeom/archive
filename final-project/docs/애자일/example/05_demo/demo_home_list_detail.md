# Demo Scenario — 홈 목록/상세 (FLOW-HOME + FLOW-DETAIL)

## 시나리오 1 — 목록 → 상세 성공
1. 홈 화면 진입 → 로딩 표시 확인(Loading)
2. 목록 표시 확인(Success)
3. 목록 항목 탭 → 상세 화면 이동
4. 상세 로딩 후 상세 정보 표시 확인(Success)

---

## 시나리오 2 — 목록 빈 상태
1. 데이터 0개 조건으로 홈 진입
2. 빈 상태 안내 + CTA 확인(Empty)

---

## 시나리오 3 — 상세 실패
1. 네트워크 끊기(또는 실패 조건 만들기)
2. 항목 탭 → 상세 진입
3. 에러 메시지 + 재시도 확인(Error + Retry)

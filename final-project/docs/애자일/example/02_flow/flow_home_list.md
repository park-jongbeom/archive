# User Flow — 홈 목록 (FLOW-HOME)

## 목표
- 사용자가 홈에서 핵심 콘텐츠 목록을 보고, 항목을 선택해 상세로 이동한다.

## 시작 조건
- 앱 실행 후 홈 화면에 진입할 수 있다.

## 주요 단계
1. 사용자가 홈 화면에 진입한다.
2. 시스템은 목록을 로딩한다(Loading).
3. 성공 시 목록을 표시한다(Success).
4. 목록이 0개면 빈 상태 안내와 다음 행동을 제공한다(Empty).
5. 실패 시 에러 메시지와 재시도 경로를 제공한다(Error + Retry).
6. 사용자가 항목을 탭하면 상세로 이동한다.

## 종료 조건
- 사용자가 상세 화면으로 이동

## 예외/상태(최소)
- 네트워크 오류/서버 오류
- 빈 목록(Empty)
- 중복 요청(새로고침 연타 등)

## 필요한 화면
- 홈(목록) 화면
- 상세 화면

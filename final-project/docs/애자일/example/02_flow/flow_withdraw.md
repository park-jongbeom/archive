# User Flow — 회원 탈퇴 (FLOW-WITHDRAW)

## 목표
- 사용자가 계정을 탈퇴한다.

## 시작 조건
- 사용자는 로그인된 상태다.
- 설정(또는 계정) 화면에서 “회원 탈퇴” 진입이 가능하다.

## 주요 단계
1. 사용자가 설정 화면에서 “회원 탈퇴”를 선택한다.
2. 시스템은 안내 문구/주의사항과 확인 UI를 보여준다.
3. 사용자가 탈퇴를 확정한다(버튼 탭).
4. 시스템은 탈퇴 요청을 수행한다(Loading).
5. 성공 시 로그아웃 처리 후 로그인(또는 온보딩) 화면으로 이동한다(Success).
6. 실패 시 에러 메시지와 재시도 경로를 제공한다(Error + Retry).

## 종료 조건
- 성공: 로그인(또는 온보딩) 화면으로 이동
- 실패: 탈퇴 확인 화면에서 재시도 가능

## 예외/상태(최소)
- 네트워크 오류/서버 오류
- 중복 제출(연타)
- 인증 만료(재로그인 유도)

## 필요한 화면
- 설정/계정 화면
- 탈퇴 확인 화면(다이얼로그/바텀시트 가능)
- 로그인(또는 온보딩) 화면

# User Flow — 로그인 (FLOW-LOGIN)

## 목표
- 사용자가 로그인하여 홈으로 진입한다.

## 시작 조건
- 사용자는 로그인되지 않은 상태다.
- 로그인 화면에 진입할 수 있다.

## 주요 단계
1. 사용자가 이메일/비밀번호를 입력한다.
2. 시스템은 입력값을 검증하고, 로그인 요청을 보낸다(Loading).
3. 로그인 성공 시 홈 화면으로 이동한다(Success).
4. 로그인 실패 시 에러 메시지와 재시도 경로를 제공한다(Error + Retry).

## 종료 조건
- 성공: 홈 화면 진입
- 실패: 로그인 화면에서 재시도 가능

## 예외/상태(최소)
- 입력값 오류(이메일 형식, 비밀번호 길이)
- 네트워크 오류/서버 오류
- 중복 제출(연타)

## 필요한 화면
- 로그인 화면
- 홈 화면

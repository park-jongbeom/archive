# User Flow — 상세 화면 (FLOW-DETAIL)

## 목표
- 사용자가 목록에서 선택한 항목의 상세 정보를 확인한다.

## 시작 조건
- 홈 목록에서 항목을 선택했거나, 바로가기 링크로 상세 진입이 가능하다.

## 주요 단계
1. 사용자가 상세 화면에 진입한다.
2. 시스템은 상세 데이터를 로딩한다(Loading).
3. 성공 시 상세 정보를 표시한다(Success).
4. 데이터가 없으면 빈 상태 안내를 표시한다(Empty, 필요 시).
5. 실패 시 에러 메시지와 재시도 경로를 제공한다(Error + Retry).
6. (선택) 사용자가 뒤로 가기/목록으로 돌아가기를 수행한다.

## 종료 조건
- 사용자가 뒤로 이동하거나 다른 화면으로 전환

## 예외/상태(최소)
- 네트워크 오류/서버 오류
- 존재하지 않는 항목(404/삭제됨 등 → Empty 또는 Error 정책)
- 중복 요청(재시도 연타)

## 필요한 화면
- 상세 화면
- 홈(목록) 화면

# User Flow — 지도에서 쪽지 찾아 읽기 (FLOW-MAP-READ)

## Flow ID

- `FLOW-MAP-READ`

## 목표

- 로그인한 사용자가 지도에서 주변 쪽지를 탐색하고, 쪽지 내용을 열람한다.

## 시작 조건

- 로그인됨(`F-USER-03`).
- 지도 화면 진입 가능.

## 주요 단계

1. 위치 권한 요청·응답 처리(`F-MAP-01`).
2. 현재 위치 조회(`F-MAP-02`).
3. 주변 쪽지 탐색·지도 초기 상태·개수 표시(`F-MAP-03`, `F-MAP-07`, `F-MAP-08`).
4. 마커 표시 및 클릭으로 쪽지 요약/상세 진입(`F-MAP-04`, `F-MAP-05`).
5. 필요 시 탐색 버튼으로 재조회(`F-MAP-06`).
6. 쪽지 열람 화면에서 내용 확인(`F-NOTE-02`).

## 종료 조건

- 성공: 특정 쪽지 내용을 열람 완료.
- 실패: 권한 거부·네트워크 오류 시 안내 및 설정/재시도 경로.

## 예외/상태(최소)

- 권한 거부(Error + 설정 이동 안내)
- 위치 불가·타임아웃
- 주변 쪽지 없음(Empty)
- 지도/데이터 로딩(Loading)
- 중복 요청·연타

## 관련 기능 ID

- `F-USER-03`, `F-MAP-01` ~ `F-MAP-08`, `F-NOTE-02`

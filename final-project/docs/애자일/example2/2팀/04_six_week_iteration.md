# 6주 일정(Iteration) — 2팀 (JoopJoop)

기능 정의서 **기능 목록 표의 모든 ID**를 Week1~Week6에 배치했다.

**Week6 원칙**: Freeze·QA·회귀·데모. `F-USER-07`은 명세상 불명확하므로 AC 확정 전까지 착수 제한(`06_decision_log.md`).

| 주차 | 초점 | 포함 기능 ID |
| --- | --- | --- |
| **Week1** | Foundation — 가입·로그인·로그아웃 | `F-USER-01`, `F-USER-02`, `F-USER-03`, `F-USER-04` |
| **Week2** | FLOW-MAP-READ 핵심 — 지도·권한·위치·마커·탐색 | `F-MAP-01`, `F-MAP-02`, `F-MAP-03`, `F-MAP-04`, `F-MAP-05`, `F-MAP-06`, `F-MAP-07`, `F-MAP-08` |
| **Week3** | FLOW-NOTE-WRITE + 열람 — 작성·열람·사진 | `F-NOTE-01`, `F-NOTE-02`, `F-NOTE-08` |
| **Week4** | 쪽지 반응 — 좋아요·스크랩·상태 조회 | `F-NOTE-03`, `F-NOTE-04`, `F-NOTE-11`, `F-NOTE-12` |
| **Week5** | 쪽지 관리·필터·계정 — 수정·삭제·스크랩삭제·필터·탈퇴·알림설정 | `F-NOTE-05`, `F-NOTE-06`, `F-NOTE-07`, `F-NOTE-09`, `F-USER-05`, `F-USER-06` |
| **Week6** | Final — 마이 전반·조회수·알림탐색(정리 후) | `F-USER-07`, `F-NOTE-10`, `F-MY-01`, `F-MY-02`, `F-MY-03`, `F-MY-04`, `F-MY-05` |

## 주차별 데모 권장

- **Week1**: 회원가입(또는 로그인) → 로그아웃.
- **Week2**: 권한 → 지도 로딩 → 마커·재조회.
- **Week3**: 쪽지 작성 → 지도에서 열람 → (사진) 첨부.
- **Week4**: 좋아요·스크랩 동작.
- **Week5**: 수정·삭제·필터·탈퇴/알림 중 팀이 고른 시나리오.
- **Week6**: 마이페이지 전체 + 회귀.

## 맵핑 검증

- `F-USER-01` ~ `F-USER-07`, `F-MAP-01` ~ `F-MAP-08`, `F-NOTE-01` ~ `F-NOTE-12`, `F-MY-01` ~ `F-MY-05` — 모두 상단 표 기준 포함.

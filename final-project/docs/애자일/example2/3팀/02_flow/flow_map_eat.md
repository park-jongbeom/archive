# User Flow — 지도에서 식당 탐색 (FLOW-MAP-EAT)

## Flow ID

- `FLOW-MAP-EAT`

## 목표

- 사용자가 GPS를 기반으로 식당 데이터를 불러와 목록과 상세 정보를 확인한다.

## 시작 조건

- 로그인 상태(또는 팀 정책상 게스트 허용 시 명시).
- 지도 화면 진입 가능.

## 주요 단계

1. GPS·위치 권한 처리 — `F-Tasty-15`.
2. 식당 데이터 불러오기 — `F-Tasty-16`.
3. 식당 정보 목록 표시 — `F-Tasty-17`.
4. 식당 상세 표시 — `F-Tasty-18`.
5. (Should) 지도 검색·탐색 확장 — `F-Tasty-19`, `F-Tasty-20`.

## 종료 조건

- 성공: 특정 식당 상세까지 확인.
- 실패: 권한 거부·API 실패 시 안내·재시도.

## 예외/상태(최소)

- 권한 거부(Error + 설정 안내)
- 위치 미수신
- 결과 없음(Empty)
- 로딩·timeout

## 관련 기능 ID

- `F-Tasty-15`, `F-Tasty-16`, `F-Tasty-17`, `F-Tasty-18`

## 비고

- 명세 본문에 Map 전용 `##` 절이 없으므로 **통합 표**가 기준. API/데이터 소스는 Issue·Decision Log에만 2~5줄로 고정.

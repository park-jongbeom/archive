# User Flow — 피드 읽기·쓰기·댓글 (FLOW-FEED)

## Flow ID

- `FLOW-FEED`

## 목표

- 로그인한 사용자가 피드 목록을 보고 글을 작성하며, 상세에서 댓글까지 남긴다.

## 시작 조건

- 로그인됨(`F-Tasty-02`).
- 홈/피드 화면 진입 가능.

## 주요 단계

1. 피드 목록 조회(로딩·빈·에러) — `F-Tasty-04`, `F-Tasty-07`.
2. 글 작성 화면 진입 → 입력·검증 → 등록 — `F-Tasty-05` (식당 검색·사진·별점은 Should).
3. 목록에서 항목 선택 → 상세 — `F-Tasty-08`.
4. 댓글 입력·등록 — `F-Tasty-09` (글자 수 등 명세 제한은 AC로 고정).

## 종료 조건

- 성공: 댓글이 상세에 반영되거나, 작성 글이 목록에 보인다.
- 실패: 검증·네트워크 오류 시 안내.

## 예외/상태(최소)

- 빈 피드(Empty)
- 로딩·timeout(Error + Retry)
- 본문·댓글 길이 제한
- 중복 제출 방지

## 관련 기능 ID

- `F-Tasty-02`, `F-Tasty-04`, `F-Tasty-05`, `F-Tasty-07`, `F-Tasty-08`, `F-Tasty-09`

## 비고

- 좋아요 `F-Tasty-10`, 정렬/필터 `F-Tasty-12`~`13`은 플로우 보완 단계에서 확장.

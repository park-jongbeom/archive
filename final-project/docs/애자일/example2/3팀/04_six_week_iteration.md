## 주차별 데모 권장

- **Week1**: 가입·로그인·로그아웃.
- **Week2**: 피드 목록·글 작성(식당 검색 포함 가능 범위).
- **Week3**: 상세·댓글·정렬/필터·사진 중 팀이 고른 **한 줄 시나리오**.
- **Week4**: 지도 → 식당 목록 → 상세.
- **Week5**: 지도 탐색 + 마이 화면 일부.
- **Week6**: 테이스티 리스트 시나리오 + 전체 회귀(피드·지도).

# User Flow — 공부 세션 완주 (FLOW-STUDY)

## Flow ID

- `FLOW-STUDY`

## 목표

- 로그인한 사용자가 홈에서 학습 화분을 준비하고, 공부 세션을 측정·종료·기록까지 완료한다.

## 시작 조건

- 앱에서 로그인된 상태(`F-Auth-03` 또는 `F-Auth-02`).
- 홈 탭 진입 가능(`F-Home-01`).

## 주요 단계

1. 홈에서 닉네임·화분·오늘 날짜 등 정보를 확인한다(Loading → Success).
2. 화분이 없으면 안내(Empty/토스트) 후 새 학습 기록(화분) 생성 화면으로 이동해 생성한다(`F-Home-03`).
3. 화분 목록에서 대표 화분을 선택하거나 생성 후 홈으로 돌아온다(`F-Home-02`).
4. “공부 시작”에 진입해 스톱워치로 시간을 측정한다(`F-Study-01` — 일시정지/재개 포함).
5. 종료 시 다이얼로그에서 학습 기록(메모)을 입력·저장한다(`F-Study-03`).
6. 결과 화면에서 요약을 확인하고, 필요 시 이미지 저장 등 후속 행동을 한다(`F-Study-04`).

## 종료 조건

- 성공: 결과 화면까지 도달하고 기록이 저장된 상태.
- 실패: 네트워크/권한 등으로 저장 불가 시 사용자에게 메시지와 재시도 경로 제공.

## 예외/상태(최소)

- 홈 데이터 로딩 실패(Error + Retry), 목표 없음(Empty + CTA).
- 이미지 로딩 실패 시 기본 아이콘(명세).
- 공부 중 백그라운드 복귀 시 시간 복구(명세).
- 종료 다이얼로그 취소 시 공부 화면 복귀.
- 중복 제출(연타) 방지.

## 관련 기능 ID (명세 참조)

- `F-Auth-02`, `F-Auth-03`, `F-Home-01`, `F-Home-02`, `F-Home-03`, `F-Study-01`, `F-Study-03`, `F-Study-04`

## 비고

- `F-Study-02`(함께 공부)는 본 Flow의 필수 단계가 아니다. 별도 실험·백로그.

# Demo Scenario — FLOW-STUDY (성공)

1. 로그인 화면에서 이메일·비밀번호 입력 후 로그인 탭 → 로딩 표시 → 홈 진입.
2. 홈에서 화분 정보 확인 → (필요 시) 새 화분 생성 완료 후 홈 복귀.
3. 공부 시작 → 스톱워치 동작 확인 → 일시정지·재개 1회.
4. 학습 종료 → 기록 입력 → 저장 → 결과 화면 표시.
5. (선택) 결과 화면에서 이미지 저장 시도 → 권한 플로우.

# Demo Scenario — FLOW-STUDY (실패/상태)

1. 홈 진입 직후 네트워크 끊김(또는 비행기 모드) → 에러 메시지·재시도.
2. 화분 없음 상태에서 Empty/토스트 확인 후 생성 플로우로 유도.
3. 종료 다이얼로그에서 취소 → 공부 화면 복귀·타이머 재개.

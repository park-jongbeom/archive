### 4-3. 양식 보완 필요 사항 (Phase 5 진입 전 결정)

#### B1. 사전 점검 자동화 명령을 v2 양식 §🚨 즉시 조치 위에 명시
- 현재 v2는 [risk_taxonomy.md §4](../../../teams-docs/.shared/risk_taxonomy.md) 참조만 함
- 보조강사가 매번 명령 찾는 시간 추가 → **v2 §1 AM 양식 상단에 R13/R14 사전 점검 명령 1줄씩 직접 명시** 권장
- 영향: v2 양식 +10줄 (작성 시간 영향 거의 없음, 누락 방지 효과 큼)

#### B2. 회의 시간 부족 시 의제 분할 룰 명시
- 현재 v2는 3건 압축 룰만. 4번째·5번째 의제는 자동 이월
- 시뮬레이션에서 4·5번째 (부팀장/Must 압축)가 계속 이월되는 패턴 확인
- 보완: "**carry-over 2회+ 이월 시 강사에게 별도 1:1 통지 권유**" 룰을 §0 사용 흐름에 추가

#### B3. 강사 사전 통지 1순위가 "회의 첫 안건"으로 명시
- 시뮬레이션에서 Case 2/3의 결정적 효과 = "회의 첫 안건"으로 강사가 시작한 것
- 보완: v2 §🚨 강사 사전 통지 의제 1번 옆에 "(회의 첫 안건 권유)" 명시

#### B4. 사전 통지 채널 timeout 명시
- 현재 "회의 5분 전" 만. 강사가 그 시점에 채팅 못 보면 무효
- 보완: "**최소 회의 시작 10분 전 + 강사 본인 알림 확인 응답 받기**" 룰

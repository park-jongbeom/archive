# 1팀 일일 스냅샷

> 매일 일일 진척 체크([`../review/daily_check_guide.md`](../review/daily_check_guide.md)) 수행 후 결과를 이곳에 저장합니다.

## 파일명 규칙

`YYMMDD.md` — 예: `260513.md` = 2026년 5월 13일

## 형식

[`../review/daily_check_guide.md` § 8 출력 템플릿](../review/daily_check_guide.md#8-%EC%B6%9C%EB%A0%A5-%ED%85%9C%ED%94%8C%EB%A6%BF-%EB%B3%B4%EA%B3%A0-%ED%98%95%EC%8B%9D) 그대로.

## 활용

다음 체크 시 직전 스냅샷을 읽어 "어제 대비" 변화량을 산출합니다. 주말은 건너뛰어도 무방하며, 월요일에 "지난 금요일 대비"로 비교합니다.

## 보관 정책

- 모든 스냅샷 영구 보관 (6주 프로젝트 종료 후 회고용)
- 주차 종료 시 [`../review/weekN_summary.md`](../review/) 로 주간 요약본 별도 생성 권장

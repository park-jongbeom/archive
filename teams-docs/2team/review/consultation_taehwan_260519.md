# 김태환 1:1 상담일지 (2026-05-19)

> **일시**: 2026-05-19 오전 / 약 26분
> **참여**: 김태환 (2팀), 보조강사

---

파이널 프로젝트 진행 회고

- 본인 고민을 두 가지로 정리하여 안건을 명확히 제시함(① 구체화 어려움, ② AI 활용·학습 시간 부족).
- 작성한 플로우 문서조차 충분히 구체화되지 않은 상태로 평가하며, 현재는 정원화 부팀장이 앞서서 구체화해 주고 있어 분담 단위가 잡히지 않는 점을 힘들어함.
- 중간 프로젝트에서 정재훈 훈련생에게 구체화를 직접 가이드해 주었던 경험을 회상하며 그 과정 자체의 부담을 자술함.

학습·작업 환경 현황

- AI 활용은 초안 → 본인 분석 → 수정 → 100% 이해 후 넘기는 사이클을 본인 원칙으로 유지하나, 현재는 새 작업이 계속 내려와 학습 사이클이 차단되는 패턴을 자술함.
- 본인이 임무를 떠맡게 된 경위는 "할 수 있을 것 같아서"라는 능동적 선택에 가깝다고 자술.

일정·페이스 관련

- 보조강사가 일정 산정 시 1.5배 보정, MVP 우선 배포, 주말 일정 사용 금지 원칙을 전달함.
- 본인은 단기 프로젝트임을 감안해 1.2~1.3 보정 계수로 적용하겠다고 수용함.
- 프로젝트 시작 시 본인이 미리 공지했던 1~2일 휴가 사용 룰을 다시 확인하며 적용 시점을 고려 중임.
- 2차 개발 기간 정의는 미합의 상태이며, 보조강사가 회의 자리에서 직접 의제로 제기하기로 함.

# 1:1 상담일지 양식 (보조강사 기록용)

## 작성 원칙

- 1쪽 안에서 끝낸다. 한 섹션당 불릿 2~4개를 목표로 함.
- 객관 사실 + 본인 자술 요지 위주. 직접 인용은 가급적 피하고 정리체로 옮긴다.
- 번호 없는 불릿(`-`)으로 나열. 한 줄당 한 사실/한 자술.
- 분석·해석·후속 hooks·종료 발화 같은 부가 섹션은 두지 않는다. 그런 내용은 별도 `advisory_*_session_summary.md`로 분리.
- 희망/기피 등 민감 정보는 본인이 명시한 경우에만 기록.

## 파일명 규칙

```
consultation_{영문이름}_{YYMMDD}.md
```

같은 날 여러 회차일 경우 `_am` / `_pm` 접미.

## 표준 섹션 구성

상담 시점에 따라 아래 3섹션을 기본으로 하고 필요 시 행정 섹션을 추가한다.

| 시점 | 권장 섹션 |
|---|---|
| 프로젝트 종료 직후 / 다음 프로젝트 직전 | 중간 프로젝트 회고 / 학습 현황 / 다음 프로젝트 관련 의견 및 팀 구성 희망 |
| 프로젝트 진행 중 | 진행 회고 / 학습·작업 환경 현황 / 일정·향후 운영 |
| 행정 사안 발생 시 | (위 섹션) + 행정 관련 문의 및 처리 결과 |

## 본문 템플릿

```markdown
# {이름} 1:1 상담일지 ({YYYY-MM-DD})

> **일시**: {YYYY-MM-DD} {AM/PM} / 약 {분}분
> **참여**: {이름} ({팀번호}팀), 보조강사

---

{섹션 1 제목}

- {본인 자술 요지 1}
- {본인 자술 요지 2}
- {보조강사 관찰 추가 맥락}

{섹션 2 제목}

- {학습 만족도/난이도}
- {본인 적용 방법}

{섹션 3 제목}

- {본인 목표}
- {팀 구성/협업 희망}
  - 희망 대상: {이름}
  - 기피 대상: {이름}

행정 관련 문의 및 처리 결과 (해당 시)

- {문의 내용}
- {처리 경로/결과}
```

## 자매 양식과의 차이

| 양식 | 용도 |
|---|---|
| `consultation_*.md` (본 양식) | 정돈된 상담 결과 기록, 행정·공유 가능 |
| `advisory_*_session_summary.md` | 코칭 흐름·코호트 케어 추적용 상세 기록 |
| `*_instructor_brief.md` | 강사진 공유용 1쪽 요약 |

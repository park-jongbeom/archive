## 0. 박종범 표준 테스트 매트릭스

각 항목 진입 시 **항상** 다음 7-step 점검을 1단위로 수행:

| Step | 점검 | 이유 |
|---|---|---|
| **S1. 디바이스** | Note 10+ (AOS12) + Z Fold 4 (AOS16) 양쪽 | OS 버전 + 폼팩터 동시 커버 |
| **S2. 세로 모드** | 기본 회전 잠금 상태 동작 | 시연 메인 시나리오 |
| **S3. 가로 모드** | 회전 후 동일 시나리오 재수행 | 레이아웃 회귀·잘림 검출 (Z Fold4는 펼친 상태 가로 시 네비게이션 바 겹침 주의) |
| **S4. 폰트 max** | 시스템 설정 → 디스플레이 → 글자 크기 최대 | 텍스트 잘림·버튼 영역 잘림 검출 |
| **S5. 상태 보존** | 화면 진입 → 회전 → 동일 데이터·동의 상태 유지 여부 | 회전 시 상태 리셋 회귀 (인증_006 약관 동의 풀림 사례) |
| **S6. 연속 동작** | 진입 → 뒤로 → 재진입 / 취소 → 재시도 | 토스트 잔존·이벤트 채널 1회성 회귀 (인증_001 사례) |
| **S7. Fix 회귀** | 이미 fix 된 commit 재검증 | 다른 변경에 의한 회귀 검출 |
